# 3rd Yr Week 2 Module - DSA with Java

This repository contains Java implementations of fundamental Searching and Sorting algorithms as part of the Week 2 DSA Module. Each problem is solved using a real-world scenario to demonstrate the practical application of Data Structures and Algorithms.

## Programs Included

1. Fraud Transaction Detection – Linear Search
2. Server Log Search – Binary Search
3. Sort Employee Salaries – Bubble Sort
4. Sort Website Response Time – Insertion Sort
5. Prioritize Bug Reports – Selection Sort

## Concepts Covered

- Linear Search
- Binary Search
- Bubble Sort
- Insertion Sort
- Selection Sort
- Time Complexity Analysis
- Space Complexity Analysis
- Java Programming Fundamentals

## Technologies Used

- Java
- VS Code / IntelliJ IDEA
- Git & GitHub

## Learning Outcomes

- Understand searching techniques and their applications.
- Compare different sorting algorithms.
- Analyze algorithm efficiency using time and space complexity.
- Strengthen problem-solving skills using Java.

### Week 2 Module PDF

The uploaded PDF contains the complete Week 2 DSA assignment, including handwritten notes explaining the logic and approach of each algorithm, Java source code implementations, VS Code screenshots of the programs, output screenshots, and complexity analysis. It acts as a comprehensive study and submission document for the module covering Linear Search, Binary Search, Bubble Sort, Insertion Sort, and Selection Sort.
